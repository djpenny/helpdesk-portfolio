# New Employee Laptop Setup SOP

## Purpose
Standard procedure for preparing a new employee laptop.

## Steps

1. Unbox and inspect hardware
2. Install Windows updates
3. Install required software:
   - Microsoft 365
   - Browser
   - VPN client
4. Join device to domain (if needed)
5. Configure Outlook, Teams, OneDrive
6. Ensure antivirus is installed and updated
7. Test:
   - Wi-Fi  
   - Printer  
   - Login  
   - Email  
   - Browser  

## Outcome
A fully configured workstation ready for employee use.
