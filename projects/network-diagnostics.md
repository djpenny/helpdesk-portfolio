# Network Diagnostics

## Tools Used
- Command Prompt  
- Router admin page  
- Windows Network Troubleshooter  

## Commands Demonstrated

### Check IP address:
```
ipconfig
```

### Test connectivity:
```
ping google.com
```

### Trace route:
```
tracert google.com
```

### Reset network components:
```
netsh int ip reset
netsh winsock reset
```

## What This Demonstrates
- Ability to diagnose slow or failing connections  
- Understanding how packets move through the network  
- Basic Windows networking fluency
