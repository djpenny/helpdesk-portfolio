# Active Directory – User Creation Project

##  Goal
Demonstrate a realistic helpdesk task: creating a new user in Active Directory.

##  Tools Used
- Windows Server (Lab)
- Active Directory Users and Computers
- Azure AD (optional)

##  Steps Performed

1. **Opened Active Directory Users and Computers**
2. Navigated to the correct **OU (Organizational Unit)**
3. Right-clicked → *New* → *User*
4. Entered:
   - First Name  
   - Last Name  
   - Username (samAccountName)
5. Set a temporary password & required password reset
6. Added user to groups:
   - Domain Users  
   - Office Staff (example)  
7. Verified account properties:
   - Logon hours  
   - Profile  
   - Member of  

##  What I Learned
- Understanding user attributes  
- Group membership and permissions  
- Realistic workflow for onboarding a new employee
